# BGP高级配置示例 v1.13

## 概述

本示例展示了如何使用IPv6 WireGuard Manager v1.13的高级BGP配置功能，包括Router ID、AS Number、Neighbors、Passwords、Multihop等参数的配置。

## 配置场景

假设我们有一个IPv6 WireGuard VPN服务器，需要配置BGP路由来分发IPv6前缀。

### 网络拓扑

```
Internet
    |
    | (BGP)
    |
[Upstream Provider]
    | ASN: 65000
    | IPv6: 2001:db8::1/128
    |
[Our Server]
    | ASN: 65001
    | Router ID: 192.168.1.1
    | IPv6: 2001:db8:1::1/128
    |
[WireGuard Clients]
    | IPv6: 2001:db8:1::/64
```

## 配置步骤

### 1. 启动BGP配置

```bash
# 运行主程序
sudo ./ipv6-wireguard-manager.sh

# 选择 "6. BGP配置管理"
# 选择 "1. 交互式BGP配置"
```

### 2. 配置Router ID

```
请输入BGP Router ID (默认: 192.168.1.1): 192.168.1.1
```

**说明**: Router ID用于BGP会话标识，通常使用服务器的IPv4地址。

### 3. 配置AS Number

```
请输入BGP AS Number (默认: 65001): 65001
```

**说明**: AS Number是自治系统号，用于标识我们的网络。

### 4. 配置Upstream ASN

```
请输入上游ASN (默认: 65000): 65000
```

**说明**: 上游ASN是提供商的自治系统号。

### 5. 配置Neighbors

```
请输入BGP邻居数量 (默认: 1): 1

配置邻居 1:
  邻居IPv6地址: 2001:db8::1
  邻居AS号: 65000
  密码: mypassword123
  描述: Upstream Provider
```

**说明**: 
- 邻居IPv6地址：上游提供商的BGP对等体地址
- 邻居AS号：上游提供商的AS号
- 密码：BGP会话认证密码
- 描述：便于识别的描述信息

### 6. 配置Multihop

```
是否启用多跳BGP? (y/n, 默认: n): n
```

**说明**: 多跳BGP用于非直连的BGP会话。

### 7. 配置IPv6 Prefixes

```
请输入IPv6前缀数量 (默认: 1): 2

配置前缀 1:
  IPv6前缀: 2001:db8:1::/64

配置前缀 2:
  IPv6前缀: 2001:db8:2::/64
```

**说明**: 这些前缀将通过BGP分发给客户端。

## 生成的BIRD配置

配置完成后，系统会生成以下BIRD配置文件：

### BIRD 2.x配置 (`/etc/bird/bird.conf`)

```bird
# BIRD 2.x Configuration
# Generated by IPv6 WireGuard Manager v1.13

router id 192.168.1.1;

protocol device {
    scan time 10;
}

protocol kernel {
    ipv6 {
        import all;
        export all;
    };
    learn;
    scan time 20;
}

protocol static {
    ipv6;
    route 2001:db8:1::/64 via "wg0";
    route 2001:db8:2::/64 via "wg0";
}

protocol bgp upstream {
    local as 65001;
    neighbor 2001:db8::1 as 65000;
    password "mypassword123";
    multihop 1;
    
    ipv6 {
        import all;
        export all;
    };
    
    hold time 180;
    keepalive time 60;
    connect retry time 120;
    start delay time 5;
    error wait time 60, 300;
    error forget time 300;
    disable after error;
    next hop self;
}
```

## 验证配置

### 1. 检查BIRD配置语法

```bash
# 检查BIRD 2.x配置
sudo bird -c /etc/bird/bird.conf -p

# 检查BIRD 3.x配置
sudo birdc configure check
```

### 2. 启动BIRD服务

```bash
# 启动BIRD
sudo systemctl start bird

# 检查状态
sudo systemctl status bird

# 启用开机自启
sudo systemctl enable bird
```

### 3. 验证BGP连接

```bash
# 查看BGP协议状态
sudo birdc show protocols

# 查看BGP邻居状态
sudo birdc show protocols all upstream

# 查看路由表
sudo birdc show routes

# 查看BGP路由
sudo birdc show route protocol upstream
```

## 高级配置选项

### 1. 多邻居配置

```bash
# 配置多个BGP邻居
BGP_NEIGHBORS="2001:db8::1,65000,password1,Provider1;2001:db8::2,65001,password2,Provider2"
```

### 2. 路由过滤

```bird
# 在BIRD配置中添加路由过滤
filter bgp_in {
    if net ~ [2001:db8::/32] then accept;
    reject;
}

protocol bgp upstream {
    # ... 其他配置 ...
    ipv6 {
        import filter bgp_in;
        export all;
    };
}
```

### 3. 路由聚合

```bird
# 配置路由聚合
protocol static {
    ipv6;
    route 2001:db8::/32 via "wg0";
}
```

## 监控和维护

### 1. 监控BGP状态

```bash
# 创建监控脚本
cat > /usr/local/bin/bgp-monitor.sh << 'EOF'
#!/bin/bash
echo "=== BGP Status ==="
sudo birdc show protocols
echo ""
echo "=== BGP Routes ==="
sudo birdc show routes protocol upstream
echo ""
echo "=== BGP Neighbors ==="
sudo birdc show protocols all upstream
EOF

chmod +x /usr/local/bin/bgp-monitor.sh

# 添加到crontab
echo "*/5 * * * * /usr/local/bin/bgp-monitor.sh >> /var/log/bgp-monitor.log" | sudo crontab -
```

### 2. 日志管理

```bash
# 配置BIRD日志
sudo nano /etc/bird/bird.conf

# 添加日志配置
log syslog { debug, trace, info, remote, warning, error, auth, fatal, bug };
log stderr all;
log "/var/log/bird.log" { debug, trace, info, remote, warning, error, auth, fatal, bug };
```

### 3. 性能优化

```bash
# 优化BIRD性能
sudo nano /etc/bird/bird.conf

# 调整扫描时间
protocol device {
    scan time 5;
}

protocol kernel {
    scan time 10;
}
```

## 安全考虑

### 1. 密码安全

```bash
# 使用强密码
BGP_PASSWORD="$(openssl rand -base64 32)"

# 定期更换密码
# 建议每3-6个月更换一次BGP密码
```

### 2. 访问控制

```bash
# 限制BIRD管理访问
sudo nano /etc/bird/bird.conf

# 添加访问控制
control {
    socket "/var/run/bird.ctl";
    socket "/var/run/bird6.ctl";
    socket "/var/run/bird.ctl" {
        user "bird";
        group "bird";
    };
}
```

### 3. 防火墙规则

```bash
# 配置防火墙规则
sudo ufw allow from 2001:db8::1 to any port 179
sudo ufw allow from 2001:db8::2 to any port 179
```
