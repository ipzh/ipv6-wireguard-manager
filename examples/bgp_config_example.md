# BGP配置示例

本文档展示如何使用IPv6 WireGuard Manager的BGP配置功能。

## 功能概述

新增的BGP配置功能包括以下参数设置：

- **ROUTER_ID**: 路由器ID配置
- **AS_NUMBER**: 本地AS号配置  
- **UPSTREAM_ASN**: 上游ASN配置
- **NEIGHBORS**: BGP邻居配置（支持密码和描述）
- **MULTIHOP**: BGP multihop配置
- **IPV6_PREFIXES**: IPv6前缀宣告配置

## 使用方法

### 1. 通过主菜单访问

```
主菜单 -> 6. BGP配置管理
```

### 2. 交互式配置

选择"交互式BGP配置"选项，系统会引导您完成以下配置：

#### 路由器ID配置
- 自动检测系统IP地址作为默认值
- 支持手动输入IPv4地址
- 验证IP地址格式

#### AS号配置
- 默认值：65001
- 支持范围：1-4294967295
- 自动清理非数字字符

#### 上游ASN配置
- 默认值：65000
- 用于配置上游BGP邻居

#### BGP邻居配置
格式：`名称,IP地址,AS号,密码,描述`

示例：
```
upstream1,2001:db8::1,65000,password123,主要上游ISP
peer1,2001:db8::10,65010,,对等连接1
customer1,2001:db8::100,65100,secret456,客户1
```

#### Multihop配置
- 用于非直连的BGP邻居
- 支持IPv4和IPv6邻居

#### IPv6前缀配置
- 默认值：2001:db8::/48
- 支持多个前缀（逗号分隔）
- 自动计算下一跳地址

## 配置示例

### 基本BGP配置

```bash
# 路由器ID
BGP_ROUTER_ID="10.0.0.1"

# AS号
BGP_AS_NUMBER="65001"

# 上游ASN
BGP_UPSTREAM_ASN="65000"

# BGP邻居
BGP_NEIGHBORS="upstream1,2001:db8::1,65000,password123,主要上游|peer1,2001:db8::10,65010,,对等连接1"

# Multihop
BGP_MULTIHOP="yes"

# IPv6前缀
BGP_IPV6_PREFIXES="2001:db8::/48,2001:db8:1::/48"
```

### 生成的BIRD配置

```bird
# BIRD 2.x BGP Configuration for IPv6 WireGuard
# Generated by IPv6 WireGuard Manager
# Router ID: 10.0.0.1
# AS Number: 65001
# BIRD Version: 2.x

# 路由器ID
router id 10.0.0.1;

# 设备协议
protocol device {
    scan time 10;
}

# 内核协议 - 处理路由表
protocol kernel {
    ipv6 {
        import all;
        export all;
    };
    learn;
    scan time 20;
}

# 直连协议 - 处理直连网络
protocol direct {
    ipv6;
    interface "wg0";
}

# BGP协议配置
protocol bgp {
    local as 65001;
    
    # BGP邻居配置
    # 主要上游
    neighbor 2001:db8::1 as 65000;
    password "password123";
    multihop;
    
    # 对等连接1
    neighbor 2001:db8::10 as 65010;
    multihop;
    
    ipv6 {
        import filter bgp_import;
        export filter bgp_export;
    };
    
    # 连接参数
    connect retry time 30;
    hold time 180;
    keepalive time 60;
}

# 静态路由配置
protocol static {
    ipv6;
    
    # 宣告IPv6前缀
    route 2001:db8::/48 via 2001:db8::1;
    route 2001:db8:1::/48 via 2001:db8:1::1;
}

# 过滤规则
filter bgp_import {
    # 接受所有路由
    accept;
}

filter bgp_export {
    # 导出所有路由
    accept;
}

# 日志配置 (BIRD 2.x 语法)
log syslog { debug, trace, info, remote, warning, error, auth, fatal, bug };
log "/var/log/bird/bird.log" { info, remote, warning, error, auth, fatal, bug };
```

## 注意事项

1. **权限要求**: 需要root权限运行BGP配置功能
2. **BIRD安装**: 确保已安装BIRD 2.x或3.x
3. **网络连通性**: 确保与BGP邻居的网络连通性
4. **防火墙配置**: 开放BGP端口（179/TCP）
5. **IPv6支持**: 确保系统支持IPv6

## 故障排除

### 常见问题

1. **BGP邻居无法建立连接**
   - 检查网络连通性
   - 验证AS号配置
   - 检查密码设置

2. **配置语法错误**
   - 使用"测试BGP配置"功能
   - 检查BIRD日志文件

3. **服务启动失败**
   - 检查配置文件权限
   - 验证BIRD版本兼容性

### 日志文件位置

- BIRD日志: `/var/log/bird/bird.log`
- 系统日志: `journalctl -u bird`
- 配置备份: `/etc/ipv6-wireguard/exports/`
