## 使用方式

### 快速开始
```bash
# 下载并安装
wget https://raw.githubusercontent.com/your-repo/ipv6-wireguard-manager/main/install.sh
chmod +x install.sh
sudo ./install.sh

# 运行管理器
sudo ipv6-wg-manager
```

### 模块化运行
```bash
# 使用核心版本（推荐）
sudo ./ipv6-wireguard-manager-core.sh

# 使用完整版本
sudo ./ipv6-wireguard-manager.sh
```

### 客户端安装
```bash
# Linux/Unix/macOS
wget https://your-server.com/client-installer.sh
chmod +x client-installer.sh
./client-installer.sh

# Windows
# 下载 client-installer.ps1 并运行
```
