## 🚀 部署方式

### Docker部署（推荐）
```bash
# 基础部署
docker-compose up -d

# 生产环境
docker-compose -f docker-compose.production.yml up -d

# 微服务架构
docker-compose -f docker-compose.microservices.yml up -d
```

### 系统服务部署
```bash
# 使用安装脚本
./scripts/install.sh

# 手动部署
sudo systemctl start ipv6-wireguard-manager
```
