## 🐛 故障排除

### 常见问题

#### 1. 服务启动失败
```bash
# 检查服务状态
sudo systemctl status ipv6-wireguard-manager

# 查看错误日志
sudo journalctl -u ipv6-wireguard-manager -f
```

#### 2. 客户端连接问题
```bash
# 检查WireGuard配置
sudo wg show

# 检查防火墙规则
sudo ufw status
```

#### 3. BGP连接问题
```bash
# 检查BIRD状态
sudo birdc show protocols

# 检查BGP邻居
sudo birdc show protocols all
```

### 日志分析

```bash
# 查看系统日志
sudo journalctl -u ipv6-wireguard-manager

# 查看应用日志
tail -f /var/log/ipv6-wireguard-manager/manager.log

# 查看错误日志
tail -f /var/log/ipv6-wireguard-manager/error.log
```

## 📚 高级用法

### 命令行操作

```bash
# 直接运行功能模块
sudo ipv6-wireguard-manager --module client_management
sudo ipv6-wireguard-manager --module network_management
sudo ipv6-wireguard-manager --module firewall_management
```

### 配置文件编辑

```bash
# 编辑主配置文件
sudo nano /etc/ipv6-wireguard-manager/manager.conf

# 编辑WireGuard配置
sudo nano /etc/wireguard/wg0.conf

# 编辑BIRD配置
sudo nano /etc/bird/bird.conf
sudo nano /etc/bird/bird6.conf
```

### 脚本集成

```bash
# 在脚本中调用功能
source /opt/ipv6-wireguard-manager/modules/common_functions.sh
source /opt/ipv6-wireguard-manager/modules/client_management.sh

# 调用函数
add_client "client1" "10.0.0.2" "2001:db8::2"
```
